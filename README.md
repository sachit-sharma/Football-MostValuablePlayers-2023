# Football Player Value Analysis — Big 5 European Leagues 2022-23

A data science project quantifying player performance relative to market value across the Premier League, La Liga, Bundesliga, Serie A, and Ligue 1.

**[View the interactive Tableau dashboard →](#)** *(link to be added)*

---

## Project Question

Which players delivered the most output per euro of market value in the 2022-23 season — and how does that vary by position, league, and age?

---

## Data Sources

| Source | File | Description |
|---|---|---|
| [FBref](https://fbref.com) | `data/player_stats.csv` | Season-level stats for all outfield players with ≥10 90s played. Exported via FBref's Big 5 combined stats table. |
| [Transfermarkt via davidcariboo/player-scores](https://github.com/davidcariboo/player-scores) | `data/player_valuations.csv` | Time-series market valuations per player. |
| [davidcariboo/player-scores](https://github.com/davidcariboo/player-scores) | `archive (3)/players.csv` | Player ID ↔ name bridge table used to join the two datasets. |

**Valuation anchor:** The valuation closest to **2023-05-31** (end of season, just before the summer transfer window) within the 2022-07-01 – 2023-06-30 window. This reflects what a player was worth after the full season of measured statistics.

## How to Run

### 1. Install dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### 2. Build the database

```bash
python setup_db.py
```

This loads `player_stats`, `player_values`, and `players_bridge` into `data/football.db` and prints row counts.

### 3. Create the SQL views

The views are applied automatically inside the notebook. To run them standalone:

```python
import sqlite3
conn = sqlite3.connect("data/football.db")
with open("queries.sql") as f:
    conn.executescript(f.read())
```

### 4. Run the notebook

```bash
jupyter notebook notebooks/analysis.ipynb
```

Or execute headlessly:

```bash
jupyter nbconvert --to notebook --execute notebooks/analysis.ipynb --output notebooks/analysis.ipynb
```

## Methodology

### Position-Specific Scoring

Performance is scored separately per position — there is no single formula applied across FW, MF, and DF. Cross-position ranking is intentionally not done.

| Position | Formula | Rationale |
|---|---|---|
| FW | `Goals/90s + Assists + SCA×0.3 + GCA×0.5` | Attack creation and conversion; Goals is a raw season total in the source, all other stats are already per-90 |
| MF | `Assists + PasProg + Pas3rd + SCA×0.3 + (Tkl+Int)×0.2` | Progressive ball-carrying, chance creation, defensive contribution; all stats are per-90 in source |
| DF | `Tkl + Int + Clr + Blocks + AerWon` | Defensive actions per 90; all stats are per-90 in source |

**Value Score** = `perf_score / market_value_eur_M` — output per million euros of market value.

### Combined Positions

Players listed with combined positions (e.g. `MFFW`) are classified by their **primary (first-listed) position only** using `SUBSTR(Pos, 1, 2)`. This avoids double-counting and matches FBref's own ordering convention.

### Filters Applied

- **GK excluded** at every stage
- **Minimum 10 × 90s played** — removes players with trivially small sample sizes
- **Minimum €1M market value** — applied in "best value" tables only, to prevent near-zero valuations from inflating value_score artificially. The raw data retains all players including unvalued ones.

### Known Data Limitations

- **~12% name mismatch rate:** Players whose names differ between FBref and Transfermarkt formats (accents, hyphens, romanisation) could not be matched and have NULL market values. They appear in performance rankings but are excluded from value score rankings. Notable examples: Pierre Højbjerg, Kim Min-jae, Vinicius Júnior.
- **No xG/xA:** FBref's free export does not include expected goals. Goals and assists are actual-outcome stats, which slightly advantages players who were clinical and disadvantages those who created good chances but were unlucky.
- **Valuation timing:** Transfermarkt updates valuations irregularly. The closest record to 2023-05-31 may be several months away for some players.

## Key Findings

- **Best-value forwards** tend to be players in their mid-20s at mid-table clubs — undervalued by the market relative to their output. Kvaratskhelia (Napoli) led raw FW performance in the dataset.
- **Midfield value** is heavily concentrated in La Liga and Bundesliga. Kroos led raw MF performance; best-value MFs were spread across all five leagues.
- **Defensive output** has a weak correlation with market value — defenders are consistently underpriced relative to their statistical contributions. Ben Mee and Ethan Pinnock (Brentford) featured prominently in DF value rankings.
- **Younger players (21-25)** show the highest value scores across all positions — market valuations for this band often lag behind a breakout season.
- **Aggression** is higher among defenders by nature of the role, but the highest card rates per 90 are disproportionately concentrated in Ligue 1.
